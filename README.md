# Meeting Room Server

一个基于 epoll 的轻量级会议室服务器，用于自己练手学习，支持多客户端连接、房间创建/加入、文本消息转发。

## 功能特性

- 自定义二进制帧协议，高效解析
- 边缘触发 (ET) 模式的 epoll 事件驱动
- 支持房间创建、加入、退出
- 房主退出时自动解散房间
- 房间内消息广播
- 非阻塞 I/O，支持高并发连接

## 协议帧格式

| 字段       | 大小 (字节) | 说明                  |
|-----------|------------|-----------------------|
| 起始标志   | 1          | 固定为 `'$'` (0x24)   |
| 消息类型   | 2          | 大端序，见下文类型定义 |
| ip地址   | 4          |     小端序，暂时预留  |
| 载荷长度   | 4          | 大端序，载荷字节数     |
| 载荷数据   | 变长       | 实际消息内容          |
| 结束标志   | 1          | 固定为 `'#'` (0x23)   |

### 消息类型 (MsgType)

| 类型值 | 含义                     |
|-------|--------------------------|
| 4     | 发送文本消息             |
| 5     | 接收文本消息（服务器下发）|
| 6     | 请求创建房间             |
| 7     | 请求加入房间             |
| 8     | 请求退出房间             |
| 9     | 创建房间响应（返回房间号）|
| 12    | 错误消息                 |
| 13    | 房间已关闭通知           |
| 14    | 伙伴加入通知             |
| 15    | 伙伴退出通知             |
